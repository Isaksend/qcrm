\chapter{Definitions}
%\addcontentsline{toc}{chapter}{Definitions}

Following terms are used in this work:\par\bigskip

\begin{description}[style=multiline, labelwidth=5cm, leftmargin=5.5cm]

	\item[Churn]
	The event of a customer discontinuing their commercial
	relationship with an organisation.

	\item[DSR\\]
	Design Science Research; an epistemological framework for
	information systems research that produces a software artefact as
	its primary output \citep{hevner2004}.

	\item[JWT\\]
	JSON Web Token; a compact, URL-safe representation of claims used
	for stateless API authentication. 	

	\item[MVP\\]
	Minimum Viable Product; the smallest product increment that
	delivers sufficient value to validate a hypothesis.

	\item[QazCloud\\]
	A Kazakhstani domestic cloud provider offering IaaS and PaaS
	services from data centres in Almaty and Astana.

	\item[SHAP\\]
	SHapley Additive exPlanations; a unified framework for
	interpreting individual ML model predictions using game-theoretic
	Shapley values \citep{lundberg2017shap}. 

	\item[SME\\]
	Small and Medium-sized Enterprise; typically defined as an
	organisation with fewer than 250 employees.

	\item[SaaS\\]
	Software-as-a-Service; a software delivery model in which
	applications are hosted centrally and accessed by end users via a web
	browser on a subscription basis.
